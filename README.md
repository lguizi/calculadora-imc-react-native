# Calculadora de IMC com React Native

Aplicativo simples desenvolvido com Expo e React Native para calcular o IMC.

## Como executar

```bash
npm install
npm start
```

## Funcionalidades

- Calculo do IMC a partir do peso e da altura
- Exibicao do resultado com duas casas decimais
- Classificacao do IMC
- Botao para limpar os campos
